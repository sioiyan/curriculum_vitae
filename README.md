  - 当初、既存のアプリケーションのコンポーネントを流用する予定だったが「Typescriptで描かれているがany型が多用」

「ライブラリにあまり頼らないフルスクラッチな実装多数」となっており
流用が難しい箇所が多数存在したため、それらは上記で挙げた問題を解消した上で自分で1から作成した。
